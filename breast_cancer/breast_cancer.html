Overview

This project focuses on classifying breast cancer cases as malignant or benign using machine learning. The goal is to evaluate how different feature selection techniques influence model performance on the Wisconsin Breast Cancer dataset.

Feature Selection Methods

  • L1 Regularization ‚Äì Uses Lasso regression to shrink coefficients and select the most important features.
  • RFECV (Recursive Feature Elimination with Cross-Validation) ‚Äì Iteratively removes features with the least importance until optimal performance is reached.
  • SFS (Sequential Forward Selection) ‚Äì Adds features one at a time to improve model performance based on validation score.

Model Evaluation and Comparison

After feature selection, we trained models including Logistic Regression and XGBoost to compare performance across accuracy, precision, recall, and AUC.

Additional Model Comparisons

We further evaluated tree-based models to understand how they perform with feature selection:

Random Forest Classifier

Random Forest performed well even without heavy feature selection due to its ensemble nature. Accuracy improved slightly when combined with RFECV.

XGBoost Classifier

XGBoost showed the highest AUC when paired with RFECV-selected features. Its boosting framework effectively captured complex patterns in fewer features.
